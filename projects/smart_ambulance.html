Project Overview

Traffic congestion significantly delays emergency response times. This IoT-based system detects approaching ambulances and dynamically adjusts traffic signals to prioritize emergency vehicles. Using an ESP8266 microcontroller, the system integrates a sound detection module to recognize ambulance sirens, activating a priority-based traffic light system.

The three-way junction system uses potentiometers to simulate multiple roads:

  • Junction road (highest priority) - Blocks all other roads if an ambulance is detected.
  • Parallel roads (lower priority) - Allowed simultaneous green signals if no emergency is detected.
  • FIFO-based queueing - If multiple ambulances arrive, they are managed based on a first-in, first-out algorithm.

The project runs on Wokwi for simulation and later integrates a real-time Python module to enhance detection accuracy. This system can be extended to real-world smart city implementations.
